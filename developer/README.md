# 開発向けCursorテンプレート

このディレクトリには、開発者がCursorを使い始めるためのテンプレートが含まれています。

## 含まれる内容

### AGENTS.md
- プロジェクト全体の基本方針を定義
- `AGENTS-small.md` は小規模プロジェクト向けの簡易版

### Rules（ルール）
- `backend-style.mdc` - バックエンド開発のルール
- `frontend-style.mdc` - フロントエンド開発のルール
- `testing.mdc` - テストコードのルール
- `security.mdc` - セキュリティ関連のルール

### Commands（コマンド）
- `code-review.md` - コードレビュー
- `create-commit-message.md` - コミットメッセージ作成
- `create-pr.md` - PR作成
- `refactor.md` - リファクタリング
- `run-tests.md` - テスト実行
- `setup-project.md` - プロジェクトセットアップ

### Skills（スキル）
- `nextjs-best-practices/` - Next.jsのベストプラクティス
- `database-design/` - データベース設計
- `api-design/` - API設計
- `laravel-architecture/` - Laravelアーキテクチャ

### Agents（サブエージェント）
- `audit-subagent.md` - 監査・レビュー用
- `research-subagent.md` - 技術調査用
- `migration-subagent.md` - データベースマイグレーション用
- `documentation-subagent.md` - ドキュメント生成用

## 使い方

1. このディレクトリの内容をプロジェクトルートにコピー
2. `AGENTS.md` をプロジェクトに合わせて編集
3. 必要なRules、Commands、Skills、Agentsを選択して使用

詳細はルートの [README.md](../README.md) を参照してください。
