# User Story Writing Skill

## このスキルの役割
- ユーザーストーリーの作成とレビューを支援する

## 前提知識
- ユーザーストーリーの基本形式（As a... I want... So that...）
- INVEST原則（Independent, Negotiable, Valuable, Estimable, Small, Testable）

## 判断基準
- ユーザー視点で記述されているか
- 価値（So that...）が明確か
- ストーリーが独立しているか（依存関係がないか）
- 適切な粒度か（小さすぎず、大きすぎない）
- 受け入れ基準が定義可能か

## よくある改善提案
- 技術的な実装詳細ではなく、ユーザーの価値に焦点を当てる
- 複数の価値を含む場合は分割を検討
- 受け入れ基準と連動させる
- エピックとストーリーの階層を明確にする
